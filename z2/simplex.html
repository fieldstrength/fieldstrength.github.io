simplex

Informally, an \(r\)-simplex defines a region of space enclosed by \(r+1\) points.

Definition. Let \(p_0 ... p_r\) be geometrically independent points in \(\mathbb{R}^m\) (with \(m \geq r\)). An \(r\)-simplex is

$$ \sigma_r = \langle p_0 ... p_r \rangle = \bigg\{ x \in \mathbb{R}^m \bigg | x = \sum_{i=0}^{r} c_i p_i, \ c_i \geq 0, \ \sum c_i = 1 \bigg\} $$

where geometrical independence means “that is, no (r − 1)-dimensional hyperplane contains all the r + 1 points”. (Yes, this is essentially linear independence; if we consider each point besides \(p_0\) as the vector from \(p_0\) to that point).

Faces

A face of a simplex \(\sigma_r\) is defined by selecting a subset of the defining points.

For some integer \(0 \leq q \leq r\), we have a \(q\)-simplex \(\sigma_q = \langle p_{i_0} ... p_{i_q} \rangle\).

Notation

This is indicated by writing

$$ \sigma_q \leq \sigma_r $$

If it is a proper subset of the points \(\sigma_r \neq \sigma_q\) then \(\sigma_q\) is a proper face.

$$ \sigma_q \lt \sigma_r $$

Properties

There are \(\binom{q+1}{r+1}\) \(q\)-faces of an \(r\)-simplex.
