algebra

Here we mostly mean Universal algebra, where we study certain patterns that mathematical sets (types) can have. Defined as equational laws that hold in terms of certain functions, of a particular type signature, that are defined on them.

In terms of computation, these algebraic structures can be considered interfaces that certain types may satisfy.

An invitation to universal algebra

Lets consider a progression of definitions that build upon each other.

I’ll give these definitions in the programming language Idris which has both interfaces and first class proofs.

Monoids

A magma is a set with a binary operation a -> a -> a.

A semigroup is a magma with the further property that the binary operation is associative. 

interface Magma a where
  (<>) : a -> a -> a

interface Magma a => Semigroup a where
  semigroupOpIsAssociative : (l, c, r : a) ->
    l <> (c <> r) = (l <> c) <> r

A monoid is a semigroup in which there is a special identity element. “Multiplying” with this element on the left or right is the identity function. 

interface Semigroup a => Monoid a where
  unit : a

  unitIdLeft : (x : a) ->
    unit <> x = x

  unitIdRight : (x : a) ->
    x <> unit = x

Example: lists

Lists, and similar data linear data structures, form a monoid with the unit the empty list, and concatenation as the binary operation.

Example: unit type

The trivial set with one element is also a monoid. In fact it satisfies the requirements of all structures on this page. We won’t spell this out.

Rings

The essential idea of a ring is that a set has two different monoidal operations that are related to each other by a distributive law.

Groups

A Group adds inverse to Monoid. 

interface Monoid a => Group a where
  inverse : a -> a

  inverseUnitLeft : (x : a) ->
    x <> inverse x = unit

  inverseUnitRight : (x : a) ->
    inverse x <> x = unit

  • quantum mechanics

    In technical terms, this compositionality of quantum systems is captured by saying that quantum systems are described by a rig category. In other words, a category that has a notion of “addition” and “multiplication” defined on it, such that the multiplication distributes over addition. In other words, it is a monoidal category in two compatible ways, just like how a a standard ring in algebra has addition and multiplication operations that must distribute. We can think of this a a categorification of a rig (ring without negation), i.e. a 2-rig.
