group

Definition. A group is a monoid for which every element \(x\) has an inverse element \(x^{-1}\). Meaning it satisfies

$$ x \cdot x^{-1} = x^{-1} \cdot x = e $$

Homomorphism properties

Theorem. Let \(f : G_1 \to G_2\) be a group homomorphism. Then the kernel and image of \(f\) form subgroups of \(G_1\) and \(G_2\) respectively.

\(\ker f = \big\{ x \in G_1 | f(x) = e \big\}\) is a subgroup of \(G_1\).

\(im f = \big\{ x \in f(G_1) \subset G_2 \big\}\) is a subgroup of \(G_2\).

Theorem. Fundamental theorem of homomorphism (Geometry, Topology and Physics theorem 3.1) Let \(f : G_1 \to G_2\) be a group homomorphism, then the quotient space:

$$ G_1 / \ker f \cong \mathrm{im} f $$

is a group isomorphism.

Finitely-generated groups

Here we can use notation that leans on numeric intuition. If we use \(+\) for the group operation, for \(x \in G\) we can interpret

$$ nx = x + ... + x \ \mathrm{(n \ times)} $$

Definition. If all elements of a group \(G\) can be expressed as a sum of products of some special elements \(\{x_i\}\) we say that G is generated by the \(\{x_i\}\). If \(\{x_i\}\) is a finite set we say that \(G\) is finitely generated by it.

We can also use this addition operation to define the obvious notion of linear independence.
