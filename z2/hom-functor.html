hom functor

For a locally small category \(C\),

and for any fixed object \(c \in C\),

There is a covariant functor

$$ C(c,-) : C \to \mathrm{Set} $$

defined as the set of arrows with the source object \(c\).

There is also the contravariant functor

$$ C(-,c) : C^\mathrm{op} \to \mathrm{Set} $$

defined as the set of arrows with the target object \(c\).

Properties

Profunctorial

Both hom functors considered as just manifestations of one bifunctor. Actually a profunctor.

$$ C(-,-) : C^\mathrm{op} \times C \to \mathrm{Set} $$

As natural transforms

By mapping over the opposite index to the one we consider functorial, we get two natural transformations.

$$ f_* : C(-,c) \Longrightarrow C(-,d) $$

$$ f^* : C(d,-) \Longrightarrow C(c,-) $$
