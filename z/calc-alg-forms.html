Points can’t be added; vectors can.

The book draws a distinction between points (indicated by boldface variables and rounded parentheses around columns) and vectors (boldface+arrow variables and square brackets around column vectors).

There is no sensible way to add points, but points can be subtracted to yield a position vector. This intuitively maps to the fact that quantifying positions requires specifying a priviledged base point to measure from.

Solving Equations

The main algorithm: Row reduction

Row operations

(Definition 2.1.1)

  • Multiply a row by a constant
  • Add a multiple of one row to another
  • swap rows (redundant)

Theorem 2.1.3:

Theorem. If the matrix \([A,\vec{b}]\) for the linear system of equations \(A\vec{x} = \vec{b}\) can by turned into \([A_2, \vec{b}_2]\) by row operations, then that new system of equations has the same solution set.

Definition 2.1.5

Definition. A matrix is in echelon form if

  • All rows of only zeros are at the bottom
  • In every row, the pivot – first nonzero entry – is always to the right of the pivot in the preceeding row.
  • All pivots are equal to \(1\), a pivotal 1.
  • Every column that contains a pivotal 1 has all other entries zero.

Theorem 2.1.8

Theorem. Given any matrix \(A\) there exists a unique matrix \(\tilde{A}\) in echelon form that can be obtained from \(A\) by row operations.

Proof (implementation):

For each column index \(c \in \{ 1 .. w \}\) (left to right):

  1. Search downward until you find a row \(r\) with a nonzero entry (a “pivot”).
  2. Swap this row \(r\) into position \(c\). Scale it so that the pivot is 1.
  3. Add whatever multiples of this row are needed to zero out the elements of column \(c\) for rows with indices \(\gt c\)

Definition. 2.2.2 A pivotal unknown is an unknown variable in a linear system that corresponds to a column with a pivotal 1.

Theorem. 2.2.4 Solutions to linear equations. For a system of linear equations \(A\mathbf{x} = \vec{b}\) and its echelon form \([\tilde{A}, \tilde{\vec{b}}]\).

  • If \(\tilde{\vec{b}}\) has a pivotal 1, then there are no solutions.
  • Otherwise:
    • If there is any non-pivotal unknown the system has infinitely many solutions.
    • If there is only pivotal unknowns the system has a unique solution.
